# homelab-ts

## Requirements

- Nodejs
- Yarn

## Installation

Install Pulumi CLI
<https://www.pulumi.com/docs/get-started/install>

Follow the instructs from <https://www.npmjs.com/package/@muhlba91/pulumi-proxmoxve> and install the Proxmox VE Resource Provider pulumi plugin.

Run:

```shell
yarn install
```
